# Privacy Policy

Last updated: 2026-02-15

Reading Checkpoint does not collect, transmit, sell, or share personal data.

## What data is stored

- Marker state for pages you annotate (check/attention markers and inline ranges)
- Stored locally in Firefox via `browser.storage.local`

## What data is not collected

- No analytics
- No tracking identifiers
- No account data
- No external network requests by the extension

## Data location and control

- Data stays on your device in Firefox extension local storage
- You can remove data by:
  - Using extension menu actions such as "Clear all markers on this page"
  - Removing the extension from Firefox
